# 🤖 Inteligência Artificial — UNI-BH

Repositório destinado às atividades, exercícios e projetos desenvolvidos na disciplina de **Inteligência Artificial** do **Centro Universitário UniBH**.

O objetivo deste repositório é organizar os códigos, experimentos e estudos realizados ao longo da disciplina, abordando conceitos fundamentais de IA e aplicações práticas utilizando **Python e análise de dados**.

---



# 🎯 Objetivo do Repositório

Este repositório tem como finalidade:

* Organizar as atividades da disciplina
* Registrar experimentos realizados em aula
* Documentar o aprendizado em Inteligência Artificial
* Servir como material de estudo e consulta futura
